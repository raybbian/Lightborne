Expand description

Trait for types that can be used as input to Systems.

Provided implementations are:

  • (): No input
  • In<T>: For values
  • InRef<T>: For read-only references to values
  • InMut<T>: For mutable references to values
  • Trigger<E, B>: For ObserverSystems
  • StaticSystemInput<I>: For arbitrary SystemInputs in generic contexts

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
