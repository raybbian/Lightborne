Compares a Map with a PartialReflect value.

Returns true if and only if all of the following are true:

  • b is a map;
  • b is the same length as a;
  • For each key-value pair in a, b contains a value for the given key, and PartialReflect::reflect_partial_eq returns Some(true) for the two values.
