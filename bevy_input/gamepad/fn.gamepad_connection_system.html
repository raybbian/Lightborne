Handles GamepadConnectionEvents events.

On connection, adds the components representing a Gamepad to the entity. On disconnection, removes the Gamepad and other related components. Entities are left alive and might leave components like GamepadSettings to preserve state in the case of a reconnection

Whenever a Gamepad connects or disconnects, an information gets printed to the console using the info! macro.
