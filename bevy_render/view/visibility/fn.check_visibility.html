Expand description

System updating the visibility of entities each frame.

The system is part of the VisibilitySystems::CheckVisibility set. Each frame, it updates the ViewVisibility of all entities, and for each view also compute the VisibleEntities for that view.

This system needs to be run for each type of renderable entity. If you add a new type of renderable entity, youâ€™ll need to add an instantiation of this system to the VisibilitySystems::CheckVisibility set so that Bevy will detect visibility properly for those entities.
