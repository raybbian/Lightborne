pub enum ReflectMut<'a> {
    Struct(&'a mut dyn Struct),
    TupleStruct(&'a mut dyn TupleStruct),
    Tuple(&'a mut dyn Tuple),
    List(&'a mut dyn List),
    Array(&'a mut dyn Array),
    Map(&'a mut dyn Map),
    Set(&'a mut dyn Set),
    Enum(&'a mut dyn Enum),
    Opaque(&'a mut dyn PartialReflect),
}
Expand description

A mutable enumeration of “kinds” of a reflected type.

Each variant contains a trait object with methods specific to a kind of type.

A ReflectMut is obtained via PartialReflect::reflect_mut.
