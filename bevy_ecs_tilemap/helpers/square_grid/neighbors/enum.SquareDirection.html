Enum SquareDirection

pub enum SquareDirection {
    East,
    NorthEast,
    North,
    NorthWest,
    West,
    SouthWest,
    South,
    SouthEast,
}
Expand description

The eight directions in which a neighbor may lie, on a square-like grid.

Note that isometric grids are also square-like grids. In particular, there is no difference between the grid system for square and diamond-isometric grids.
